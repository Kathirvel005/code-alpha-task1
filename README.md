# Hangman Game

A simple text-based Hangman game built using Python.

## Features

- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Input validation
- Win/Loss detection

## Concepts Used

- random module
- while loop
- if-else statements
- strings
- lists

## How to Run

1. Install Python 3
2. Clone the repository

git clone https://github.com/yourusername/hangman-game.git
cd hangman-game
